Funcionamiento de la Plataforma de Cursos

Una descripción clara y sencilla de cómo nuestra plataforma conecta a administradores y estudiantes para una experiencia de aprendizaje única.

Roles en la Plataforma

Administrador

Tiene el control total para crear cursos, subir material, generar códigos de acceso y gestionar toda la plataforma.

Estudiante

Se registra, canjea códigos para acceder a los cursos y sigue una ruta de aprendizaje guiada para obtener su certificado.

Panel de Administración

  • ✓

    Gestión de Cursos

La Experiencia del Estudiante

Un camino de aprendizaje intuitivo y enfocado en alcanzar el éxito.

Paso 1

Registro y Acceso al Curso

El estudiante se registra y usa el código único que se le ha entregado. Una vez validado, el código se desactiva y el estudiante obtiene acceso permanente al curso.

Paso 2

Aprendizaje Guiado

Para avanzar, es obligatorio ver el video completo de cada lección. Esto asegura que el material se revise antes de pasar a la evaluación.

Paso 3

Evaluación y Aprobación

Al final de cada lección, se debe aprobar un cuestionario. Si no se alcanza la nota mínima (ej. 80%), se puede reintentar las veces que sea necesario.

Emisión Automática de Certificados

Al aprobar todas las lecciones de un curso, el sistema genera automáticamente un certificado personalizado con el nombre del estudiante, el nombre del curso y la fecha. El certificado estará listo para ser descargado al instante desde su perfil.
